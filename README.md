# Gene Editing Simulation Sandbox

Simulates CRISPR-Cas9 gene editing outcomes at a population and cellular level

# Features
 - Stochastic modeling of: 
     - Delivery
     - DNA cutting
     - Homology-directed and non-homologous end joining
     - Off-target editing and mutations
     - Evolutionary population dynamics
     - Real-time parameter tuning
     - Interactive Streamlit dashboard

# How to run

'''bash
pip install -r requirements.txt
python -m streamlit run app/dashboard.py
